Windows (PCL) 安装&游玩教程

阅读此文章前，先阅读 游玩教程

Nov 21, 2025
阅读时长: 2 分钟

本篇文章将引导您在 Windows 上游玩我们的服务器。

在 Windows 上可以使用的官方/非官方启动器有很多，下文将以 PCL2 为例。

准备工作

安装 PCL2

首先，请去下载 PCL2 ，密码 pcl2

下载最新版本（时间最靠前）的zip文件（通常是PCL 正式版 x.xx.x.zip），下载到本地解压。

请一定要先解压，再运行，否则可能导致后面的步骤错误。

我们强烈建议您单独在某个不为系统盘的磁盘内新建文件夹以存放 PCL2 本体（如D:\PCL2），因为 PCL2 在运行时会产生大量的文件夹（如.minecraft）在它的统计目录，难以整理。

解压后，双击运行Plain Craft Launcher 2，此时应当显示 PCL2 图标，之后显示主页,如下图：

安装 XPlus 整合包

接下来，请前往此链接下载 XPlus 1.20.1 整合包 。

下载完毕后，直接将整合包拖入打开的 PCL2 窗口。如果正常，它会提示你正在安装 MCBBS 整合包，如下图：

如果 PCL2 提示你是否要安装 Java ，请一定点确定，否则可能导致后面的步骤错误。

安装完成后，应显示下图：

安装 OptiFine 模组

接下来前往此链接下载 OptiFine 模组 。

下载完毕后，直接将文件拖入打开的 PCL2 窗口。如果正常，它会提示你是否安装 Mod ，如下图：

点确定安装。

设置 第三方登录

安装后，点击版本设置，在侧栏选择设置，找到服务器，应为下图：

将登录方式改为第三方登录：Authlib Injector 或 LittleSkin，之后按下表填写：

栏目 内容
登录方式 第三方登录：Authlib Injector 或 LittleSkin
认证服务器（必填） https://skin.nanyuan.site/api/yggdrasil
注册链接 https://skin.nanyuan.site/auth/register
服务器名称 南原认证中心
自动进入服务器

填写完毕后应如下图：

请不要点设置为 LittleSkin，这将导致后面的步骤错误

之后直接按左上角左箭头返回主页，应显示下图：

在邮箱和密码栏目填入您在您在 注册 皮肤站 账号 时填写的电子邮箱和密码。

添加服务器

先点击启动游戏，之后等待约3到20秒，就会出现游戏窗口，显示如下图的 Forge 加载页面：

再等待一会儿，就会进入游戏主菜单，如下图：

点击多人游戏，应显示下图：

点击添加服务器，并按下表填写：

栏目 内容
服务器名称 南原
服务器地址 play.nanyuan.site
服务器资源包 询问

填写完毕后，应如下图：

点击完成，回到 多人游戏 界面，应显示下图：

点击 原 字图标即可进入服务器。
