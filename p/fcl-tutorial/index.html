Android (FCL) 安装&游玩教程

阅读此文章前，先阅读 游玩教程

Nov 23, 2025
阅读时长: 2 分钟

本篇文章将引导您在 Android 上游玩我们的服务器。

说在前面

Android 端并不适合游玩，建议在 Windows 上游玩。

Android 更适合用来挂机，或者在身边没有 Windows 的时候勉强游玩。

在 Android 上可以使用的官方/非官方启动器不是很多（大部分还都是基岩版），下文将以 FCL 为例。

准备工作

安装 FCL

首先，请去下载 FCL，请下载到本地后安装。

非常抱歉，由于123网盘的缘故，您可能需要登录/注册123网盘才能下载。还请您多多理解包容。

一般手机厂商会在识别到有apk下载完成后自动进入安装流程。

在这期间，可能会询问您很多有关 权限 的问题，请全部点允许，或者有特殊流程的，自行根据指示处理。

由于不同厂商、不同品牌的Android手机有差异，在此不做详细说明。

安装成功后，打开Fold Craft Launcher，应显示下图：

点击同意并继续，进入选择 Java 页面，如下图：

点击安装/更新，安装/更新完毕后，进入主页面，如下图：

下载游戏本体

接下来，选择左边栏从上到下第3个图标（云朵箭头图标），进入下载页面，如下图：

点击搜索，输入1.20.1

点击地球图标，进入如下图页面：

点击名称旁的下载按钮（下箭头），即可开始下载。下载完毕后，应如下图所示：

点确认后，应如下图所示：

设置 第三方登录

点击右侧的 Alex 头像，开始配置第三方登录。

点击偏右下的+ 认证服务器，输入以下内容：

1

https://skin.nanyuan.site/api/yggdrasil

点下一步后，等待一会儿，会跳出如下界面：

点确认后，应如下图所示：

点击NanYua...，创建外置账户，如下图所示：

在此填入您在 注册 皮肤站 账号 时填写的游戏内角色名和密码。

填写完毕后，等待一会儿，应显示下图：

添加服务器

现在点左侧的启动进入游戏，分别会闪过包括但不限于以下界面：

点击多人游戏，应显示下图：

点击添加服务器，并按下表填写：

栏目 内容
服务器名称 南原
服务器地址 play.nanyuan.site
服务器资源包 询问

如何在文本框中输入？

先点中文本框，再点正上方的Input，会弹出手机键盘，此时可以输入/粘贴内容。

输入完毕后，点任意非键盘处回到游戏页面。

填写完毕后，应如下图：

点击完成，回到 多人游戏 界面，应显示下图：

点击 原 字图标即可进入服务器。
